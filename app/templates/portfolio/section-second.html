Tecnologías principales

Java

Es el lenguaje en el que tengo más experiencia. Aunque principalmente lo uso para el backend, desarrollé mi TFG completamente en Java, utilizando vistas JSP para el frontend y un controlador de excepciones middleware.

El proyecto consistía en una web de gestión de stock para una tienda online, accesible solo para empleados. Permitía ver productos, su stock, detalles y filtrarlos por categorías como tecnología, higiene, herramientas, cocina, etc.

Angular

Angular es mi framework principal para el desarrollo frontend. He trabajado en múltiples proyectos escalables que incorporan características avanzadas como:

  • Control de sesión
  • Gestión de cookies
  • Control de roles
  • Medidas de seguridad
Además, tengo un dominio sólido de HTML, CSS3, JavaScript y TypeScript, lo que me permite crear aplicaciones web interactivas y eficientes.

Python y Flask

Este portfolio demuestra mis habilidades en Python y Flask, elegidos por su flexibilidad y eficiencia. He desarrollado varias aplicaciones, incluyendo una de gestión de tareas con versiones web y de escritorio.

Mi experiencia incluye:

  • Configuración de servidores VPS
  • Implementación de SSL con medidas de seguridad
  • Uso de HSTS y protocolos TLS modernos
  • Implementación de cifrados fuertes
